🔧 API使用

本服务同时提供REST API和MCP协议接口，支持程序化调用：

快速测试

# REST API
curl -X POST http://localhost:8080/api/process \
  -F "audio=@your_audio.mp3" \
  -F "model=htdemucs" \
  -F "stems=vocals,drums" \
  -F "output_format=wav" \
  -F "audio_quality=high"

# MCP协议 (JSON-RPC 2.0)
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "tools/call",
  "params": {
    "name": "separate_audio",
    "arguments": {
      "file_path": "/path/to/audio.mp3",
      "model": "htdemucs",
      "stems": ["vocals", "drums"]
    }
  }
}
